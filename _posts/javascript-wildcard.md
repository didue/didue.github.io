`JavaScript` 또는 `jQuery`에서 `querySelectorAll()`을 사용할 때<br/>
와일드카드를 이용하여 *패턴으로 탐색*하고 싶은 경우엔 아래와 같이 사용해보자!

<br/><br/>

**~로 시작하는 요소**  `^`

```
//JavaScript
document.querySelectorAll("[id^='abc']");

//jQuery
$("[id^='abc']");
```

**~로 끝나는 요소** `$`

```
//JavaScript
document.querySelectorAll("[id$='abc']");

//jQuery
$("[id$='abc']");
```

**~를 포함하는 요소** `*`

```
//JavaScript
document.querySelectorAll("[id*='abc']");

//jQuery
$("[id*='abc']");
```
